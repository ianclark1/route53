You can find out more about this project at http://www.clarke.ca/archives.cfm/category/route-53 .  This project is currently hosted at http://route53.riaforge.org , which is where you can download it.

These components allow you to use Amazon's Route 53 service from ColdFusion.  These components require the use of AWSConsole (http://awsconsole.riaforge.org).
